# Loja de Jogos Digitais – Projeto de Banco de Dados

Este projeto apresenta a modelagem de um banco de dados para uma loja de jogos digitais.

## Tecnologias utilizadas

- MySQL
- MySQL Workbench
- SQL

## Estrutura do banco

O banco contém tabelas para armazenar informações sobre:

- jogos
- categorias
- plataformas
- assinaturas

## Objetivo

Demonstrar habilidades em:

- modelagem de dados
- criação de tabelas- criação de tabelas
- relacionamentos entre tabelas
- consultas SQL
